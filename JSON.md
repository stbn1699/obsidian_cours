la notation json

```json
{}
```

Les propriétées d'un objet JSON sont représentées par des paires clés/valeur : 

```json
{ "cle": "valeur", "autre_cle": "autre valeur" }

{ "nom": "PEYRACHON", "prenom": "Nicholas"}
```

les types de données : 
	- booléen
	- numériques
	- chaine de caractères
	- tableau
	- objet
	- null
